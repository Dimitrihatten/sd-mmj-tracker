# SDMMJTracker

## South Dakota Medical Cannabis Patient Portal

A HIPAA-compliant platform for South Dakota medical cannabis patients to track allotments, find dispensaries, and manage compliance.

### Features

- **Patient Verification System**: Real-time verification with SD Department of Health
- **Allotment Tracking**: Monitor monthly cannabis limits and usage
- **Dispensary Finder**: Locate and compare local dispensaries
- **Purchase History**: Track all cannabis purchases and spending
- **Compliance Monitoring**: Automated regulatory compliance checking

### Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL with encryption
- **Security**: AES-256 encryption, CSRF protection, security headers
- **Hosting**: Vercel with Cloudflare CDN
- **Analytics**: Privacy-focused analytics

### Installation

```bash
git clone https://github.com/trillyclub/sdmmjtracker.git
cd sdmmjtracker
npm install
npm run dev
```

### Environment Variables

```bash
DATABASE_URL=postgresql://...
CSRF_SECRET=your-csrf-secret
ENCRYPTION_KEY=your-encryption-key
SD_API_KEY=your-sd-api-key
```

### API Documentation

#### Patient Verification
- `POST /api/verify-patient` - Verify medical card
- `GET /api/patient/{id}` - Get patient information

#### Allotment Tracking
- `GET /api/allotment/{patient_id}` - Get current allotment
- `POST /api/purchase` - Record new purchase

#### Dispensary Services
- `GET /api/dispensaries` - List all dispensaries
- `GET /api/dispensaries/search` - Search dispensaries

### Security

- All PHI data encrypted with AES-256
- CSRF protection on all forms
- Security headers implemented
- Regular security audits performed

### Compliance

- HIPAA compliant architecture
- South Dakota cannabis law compliant
- Regular compliance audits
